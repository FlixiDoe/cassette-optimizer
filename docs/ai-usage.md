# AI Usage

This project used AI tools during research, prompting, implementation, and repository maintenance.

## Tools Used

- Gemini 3.1 Pro (web) was used for research.
- Claude Sonnet 4.6 was used to draft and refine implementation prompts.
- ChatGPT Codex (5.5 medium) was used as the coding agent for implementation.
- Git was used for commits and pushes.

Human review and project ownership remain with the repository maintainer.
